# Aether UI 快速入门指南

## 🚀 5分钟快速入门

### 第一步：创建窗口

```cpp
#include "aether/AetherApplication.h"

int main() {
    auto app = AetherApplication::create();
    return app->run();
}
```

### 第二步：添加组件

```cpp
auto container = storage.createComponent(ComponentType::Container);
auto* entry = storage.getComponent(container);
entry->layoutResult = {0, 0, 800, 600};

auto button = storage.createComponent(ComponentType::Button, container);
auto* btnEntry = storage.getComponent(button);
btnEntry->properties.setProperty(PropertyId::Width, PropertyValue(100));
btnEntry->properties.setProperty(PropertyId::Height, PropertyValue(40));
```

### 第三步：计算布局

```cpp
engine.forceRelayout();
dispatcher.onLayoutComplete();
```

### 第四步：渲染

```cpp
renderer->beginDraw();
renderer->clear(Color::White());
renderer->fillRoundedRect(btnEntry->layoutResult, 8.0f, Color::Blue());
renderer->endDraw();
```

**完整示例**:

```cpp
#include "aether/AetherApplication.h"

int main() {
    auto app = AetherApplication::create();
    auto& storage = app->getStorage();
    auto& engine = app->getLayoutEngine();
    
    // 创建容器
    auto container = storage.createComponent(ComponentType::Container);
    storage.getComponent(container)->layoutResult = {0, 0, 800, 600};
    
    // 创建按钮
    auto button = storage.createComponent(ComponentType::Button, container);
    auto* btn = storage.getComponent(button);
    btn->properties.setProperty(PropertyId::Width, PropertyValue(120));
    btn->properties.setProperty(PropertyId::Height, PropertyValue(40));
    btn->properties.setProperty(PropertyId::Text, PropertyValue("Click Me"));
    
    // 布局
    engine.forceRelayout();
    
    return app->run();
}
```


```cpp

                                 PropertyValue(FlexDirection::Row));
```

```cpp
                                 PropertyValue(JustifyContent::SpaceBetween));
```

### 3. 垂直居中

```

```

```cpp

                                 PropertyValue(FlexDirection::Column));

```

## 🎯 常见任务

### 创建带边距的按钮

```cpp
auto button = storage.createComponent(ComponentType::Button, container);
auto* entry = storage.getComponent(button);

entry->properties.setProperty(PropertyId::Width, PropertyValue(100));
entry->properties.setProperty(PropertyId::Height, PropertyValue(40));
entry->properties.setProperty(PropertyId::MarginLeft, PropertyValue(10));
entry->properties.setProperty(PropertyId::MarginTop, PropertyValue(10));
entry->properties.setProperty(PropertyId::MarginRight, PropertyValue(10));
```

### 创建文本输入框

```cpp
auto textField = storage.createComponent(ComponentType::TextField, container);
auto* entry = storage.getComponent(textField);

entry->properties.setProperty(PropertyId::Width, PropertyValue(200));
entry->properties.setProperty(PropertyId::Height, PropertyValue(30));
entry->properties.setProperty(PropertyId::PlaceholderText, 
                             PropertyValue("输入内容..."));
```

### 创建表单布局

```cpp
// 表单容器
auto form = storage.createComponent(ComponentType::Container, parent);
auto* formEntry = storage.getComponent(form);
formEntry->layoutResult = {100, 100, 400, 300};
formEntry->properties.setProperty(PropertyId::FlexDirection, 
                                 PropertyValue(FlexDirection::Column));
formEntry->properties.setProperty(PropertyId::Padding, PropertyValue(20));

// 标签+输入框对
for (int i = 0; i < 3; i++) {
    auto row = storage.createComponent(ComponentType::Container, form);
    auto* rowEntry = storage.getComponent(row);
    rowEntry->properties.setProperty(PropertyId::FlexDirection, 
                                     PropertyValue(FlexDirection::Row));
    rowEntry->properties.setProperty(PropertyId::MarginBottom, PropertyValue(10));
    
    auto label = storage.createComponent(ComponentType::Label, row);
    storage.getComponent(label)->properties.setProperty(
        PropertyId::Width, PropertyValue(80));
    
    auto input = storage.createComponent(ComponentType::TextField, row);
    storage.getComponent(input)->properties.setProperty(
        PropertyId::FlexGrow, PropertyValue(1));
}
```

## ⚡ 性能技巧

### 1. 批量更新

```cpp
// ❌ 慢
entry1->properties.setProperty(PropertyId::Width, PropertyValue(100));
engine.forceRelayout();

entry2->properties.setProperty(PropertyId::Width, PropertyValue(100));
engine.forceRelayout();

// ✅ 快
entry1->properties.setProperty(PropertyId::Width, PropertyValue(100));
entry2->properties.setProperty(PropertyId::Width, PropertyValue(100));
engine.forceRelayout();
```

### 2. 使用脏标记

```cpp
// ❌ 总是重新布局
engine.forceRelayout();

// ✅ 只重新布局变化的组件
engine.markDirty(changedComponent);
engine.relayoutIfNeeded();
```

### 3. 避免过深嵌套

```cpp
// ❌ 深度嵌套
Container
└── Container
    └── Container
        └── Container
            └── Button

// ✅ 扁平化
Container
├── Container1
│   └── Button
└── Container2
    └── Button
```

## 🚫 常见错误

### 错误1: 组件不显示

**原因**: 尺寸为0

```cpp
// ❌ 错误
entry->layoutResult = {0, 0, 0, 0};

// ✅ 正确
entry->layoutResult = {0, 0, 100, 50};
```

### 错误2: FlexGrow不生效

**原因**: 父容器没有尺寸

```cpp
// ❌ 错误
parentEntry->layoutResult = {0, 0, 0, 0};

// ✅ 正确
parentEntry->layoutResult = {0, 0, 800, 600};
```

### 错误3: 事件不响应

**原因**: 未调用onLayoutComplete

```cpp
engine.forceRelayout();
dispatcher.onLayoutComplete(); // 别忘了这个！
```
